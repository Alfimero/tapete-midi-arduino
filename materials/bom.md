# BOM y herramientas

## Componentes principales

- 1 tapete de bateria electronica enrollable.
- 1 `Arduino Micro`.
- 1 cable USB para programacion y alimentacion.
- Cables Dupont, cable fino o jumper para el conexionado interno.
- Estaño y puntos de soldadura para derivar senales del tapete.

## Herramientas recomendadas

- Cautin o estacion de soldadura.
- Multimetro para identificar `5V`, `3.3V`, `GND` y continuidad.
- Pinzas de corte y pelacables.
- Tubo termocontraible o cinta aislante.
- Pistola de silicona o fijacion mecanica ligera para alivio de tension.

## Consumibles opcionales

- Conectores JST o headers para hacer el mod desmontable.
- Protoboard pequena para pruebas previas.
- Etiquetas para identificar pads y pedales.
